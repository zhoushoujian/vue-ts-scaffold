# Vue 2 + typescript 脚手架

vue2 + typescript + webpack5(vite4) + less + eslint  
默认使用webpack，可选用vite开发

## 基础搭建

在这里我们使用的是 [pnpm](https://pnpm.io/zh/) 来作为本项目的包管理工具。

```bash
# 安装依赖
pnpm i

# 运行
pnpm run dev

# 打包
pnpm run build

# 打包文件预览
pnpm run preview
```

> 本人开发环境
>
> node v16.13.1（大于 14 版本即可）
>
> pnpm v6.30.0
